## Filosofía del proyecto
- **Specification-Driven Development:** Toda funcionalidad parte de constitución, especificación, plan y tasks documentados en `.specify/memory/`.
- **UI minimalista e inteligente:** El home invita a preguntar qué es indevelops mediante un chat retro con respuestas heurísticas.
- **Escalabilidad futura:** El motor de chat está listo para conectarse a backends/LLM reales y el flujo Git Flow + Actions soporta releases confiables.

Actualiza esta página cuando agregues nuevas secciones o cambies la arquitectura base.

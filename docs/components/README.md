# Componentes Clave

| Componente | Ubicación | Rol |
|------------|-----------|-----|
| `Layout` | `src/components/Layout/` | Contenedor general con fondo, tipografía y spacing responsive. |
| `HeroChat` | `src/components/HeroChat/` | Hero principal con copy, badges y panel del chat. |
| `ChatInput` | `src/components/ChatInput/` | Input accesible y botón de envío para preguntas del usuario. |
| `SocialPreview` | `src/components/SocialPreview/` | Card enlazable que promociona `@in_develops` cuando el chat sugiere participar. |
| `RoadmapStrip` | `src/components/RoadmapStrip/` | Visualiza los pasos `Idea → Spec → Plan → Implementación`. |
